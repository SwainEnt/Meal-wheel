## Deploy to Netlify (5 minutes)

1. Go to **netlify.com** and sign up for a free account
2. Click **"Add new site"** → **"Import an existing project"**
3. Connect your GitHub account and upload this folder, OR
4. Use **Netlify CLI**: `npx netlify deploy --build`

## Or use Netlify Drop (easiest)

1. Go to **app.netlify.com/drop**
2. Drag and drop this entire folder onto the page
3. You'll get a live URL instantly!

Note: Netlify Drop deploys static files only. For the full React build, use GitHub import.

## Run locally

```bash
npm install
npm run dev
```
